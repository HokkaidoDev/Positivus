## 🚀 Overview
- Responsive single-page layout built with HTML and SCSS.
- Eye-catching hero section with partner logos and a strong value proposition.
- Designed for digital marketing, SEO, PPC, social media, email campaigns, content creation, and analytics.

## 🎯 Key Features
- Responsive mobile-friendly layout
- Clean branding with a bright, modern color palette
- Interactive sections for services, case studies, and process
- Visual emphasis on teamwork, transparency, and growth

## 🛠️ Built With
- HTML
- SCSS
- BEM methodology
- Semantic page structure

## ▶️ How to Run
1.  **Clone the repository:**
   ```bash
   git clone https://github.com/HokkaidoDev/Positivus.git
   ```
2. Open `index.html` in your browser.
3. If you want to modify styles, update the SCSS files in `scss/styles/` and compile them to `dist/css/style.css`.
4. The page is fully static and can be deployed to any static hosting service.

## 📌 Purpose
This project is a polished marketing landing page concept for a digital agency called **Positivus**, optimized to communicate service offerings, successful campaigns, and a professional team in a visually engaging format.
